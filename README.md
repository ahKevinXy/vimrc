VIM配置文件，使用vundle做插件配置管理。

安装：

+ git clone https://github.com/shelmesky/vimrc .vim
+ cp .vim/.vimrc ~
+ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
+ :PluginInstall




### Vim 基本快捷检

* e file -- 关闭当前编辑的文件

* w 保存

* h 或退格 左移动一个位置
* i 或空格 右移动一个位置
* j 下移一行
* k 上移一行
* gj 移动到一段内的下一行
* gk 移动到一段内的上一行
* + entrer 把光标 移至下一行第一个非空白字符

* -  把光标移动至 上一行第一个非空字符

* w 前移动一个单词
* W 移动下一个单词开头
* e 前移动 一个单词
* E 移动到下一个单词开头
* b 后移动一个单词
